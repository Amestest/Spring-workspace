타임리프 예제 4

타임리프 조건문 [1] = th:if /th:unless

		th:if${조건식}
		- 조건식이 **true** 인 경우에만
		해당 속성이 작성된 요소를 화면에 출력
		th:if${만약에 관리자가 접속했다면} = 상품 수정하기를 보여주겠다!
		
		th:unless = ${조건식}
		- 조건식이 **false**인 경우에만
		해당 속성이 작성된 요소를 화 면에 출력
		th:unless = ${만약에 비밀번호가 일치하지 않는다면} = 비밀번호 찾기 버튼 보이자!

th:if / th:unless 조건식이 변수명 / 값 하나만 작성한 경우

	if문에 작성된 값이 있으면 (값 != null) -> true
	if문에 작성된 값이 없으면 (값 == null) -> false
