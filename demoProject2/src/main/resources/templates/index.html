2. thymeleaf

		웹 및 독립 실행형 환경 모두를 위한 최신 서버 측 Java 템플릿 엔진
		-> 웹 실행 == 요청 시 포워드 되는 화면
		--> 독립 실행 == html 파일 실행(인터넷 연결 x)
		
		HTML 파일에서 th(ThymeLeaf) 속성을 이용해
		컨트롤러부터 전달받은 데이터를 이용해 동적 페이지를 만들 수 잇음
		
		Spring boot 에서는 jsp가 아닌 ThymeLeaf 사용을 권장하고 있음
		
		[사용법]
		html 에서
		< !Doctype html> 밑에 작성된
		< html xmlns:th="http://www.thymeleaf.org">작성후 사용이 가능
		http://www.thymeleaf.org 이 주소를 가져오는데 th를 사용해서 작성
		
		html xmlns:별칭="템플릿가져올 주소"
